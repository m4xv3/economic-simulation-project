# Economic Dynamics Lab (EDL)
Economic Dynamocs Lab is a simple Python economic simulation. It models households, firms, wages, prices, spending, production, GDP, inflation, and unemployment.

## Current MVP
The first version will:

- Create households
- Create firms 
- Give households money
- Let firms produce goods
- Let households buy goods
- Track GDP
- Track unemployment
- Track inflation
## Goal

Build an interactive economic simulation where users can change economic variables and see how the economy responds.
